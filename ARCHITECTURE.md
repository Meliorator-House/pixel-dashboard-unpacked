# Архитектура проекта

## Frontend

- Vite + React + TypeScript.
- Основной файл: `src/App.tsx`.
- Стили: `src/styles.css`.
- Хранение данных: `localStorage`.
- Данные: задачи, проекты, финансы, цели, привычки, фокус дня, настройки Bot API.

## Bot

- Python + aiogram.
- FastAPI API для получения событий с сайта.
- JSON-хранилище `data.json`.

## Почему сайт и бот разделены

Vercel отлично подходит для сайта.  
Python-боту нужен постоянный процесс, поэтому лучше Render/Railway/VPS.

## Что улучшить следующим шагом

1. Подключить Supabase или PostgreSQL.
2. Сделать авторизацию.
3. Сделать общую базу для сайта и бота.
4. Добавить пуш-уведомления и cron.
5. Добавить настоящую AutoCRM-аналитику.
